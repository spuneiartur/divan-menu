# Divan — Smart Digital QR Menu (mockup)

Machetă de prezentare pentru o aplicație de meniu digital prin QR.
Poziționare: **smart menu layer** peste operațiunile existente — NU un POS paralel.

**Live:** https://spuneiartur.github.io/divan-menu/

## Structură
- `index.html` — markup
- `style.css` — stiluri + sistem de teme (`clean` / `warm` / `green` / `gold`)
- `script.js` — meniu, filtre, draft de comandă, cereri ospătar
- `divan-logo.png`, `wine-tasting.mp4` — assets

## Schimbă tema
În `index.html`, pe `<body data-theme="clean">`, schimbă cuvântul: `clean` (implicit, alb universal), `warm`, `green` sau `gold`.
